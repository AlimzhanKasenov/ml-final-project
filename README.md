# Прогнозирование оттока клиентов банка на основе ансамблевых методов машинного обучения

## Описание проекта

Данный проект посвящён решению задачи прогнозирования оттока клиентов банка с использованием методов машинного обучения.

Цель проекта — построить и сравнить несколько моделей классификации, которые по характеристикам клиента банка определяют вероятность его ухода.

В проекте используются базовая модель и ансамблевые методы машинного обучения:

- Logistic Regression;
- BaggingClassifier;
- RandomForestClassifier;
- GradientBoostingClassifier;
- LightGBMClassifier.

Проект демонстрирует полный цикл работы с ML-задачей:

1. создание датасета;
2. анализ данных;
3. визуализация;
4. предобработка;
5. обучение моделей;
6. сравнение метрик;
7. подбор порога классификации;
8. анализ важности признаков;
9. формирование визуального HTML-отчёта.

## Актуальность задачи

Для банков важно заранее определять клиентов, которые могут прекратить пользоваться услугами. Если банк заранее выявляет таких клиентов, он может предложить персональные условия, улучшить сервис или провести удерживающие мероприятия.

Прогнозирование оттока помогает:

- снизить потери клиентов;
- повысить качество обслуживания;
- эффективнее использовать маркетинговые кампании;
- определить ключевые факторы, влияющие на уход клиента.

## Датасет

В проекте используется синтетический датасет на русском языке, имитирующий клиентскую базу банков.

Датасет создан искусственно, чтобы показать полный процесс машинного обучения без использования закрытых банковских данных.

Размер датасета:

- 10 000 строк;
- 21 колонка.

Целевая переменная:

- `Отток`;
- `0` — клиент остался;
- `1` — клиент ушёл.

Основные признаки:

- возраст клиента;
- пол;
- город;
- банк;
- тип клиента;
- канал обслуживания;
- кредитный рейтинг;
- срок обслуживания;
- баланс;
- количество банковских продуктов;
- наличие кредитной карты;
- активность клиента;
- количество жалоб;
- обращения в поддержку;
- использование мобильного приложения;
- сумма кредитов;
- просрочки по кредитам;
- средняя оценка сервиса.

## Используемые инструменты

В проекте используются следующие инструменты и библиотеки:

- Python — основной язык программирования;
- NumPy — генерация числовых данных;
- Pandas — работа с таблицами и анализ данных;
- Matplotlib — построение графиков;
- Seaborn — визуализация данных;
- Scikit-learn — предобработка, обучение моделей и расчёт метрик;
- LightGBM — модель градиентного бустинга;
- Joblib — сохранение обученных моделей;
- HTML/CSS — визуальный отчёт для демонстрации результатов.
